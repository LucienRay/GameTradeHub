## How to start

>注意!!!<br>
>請先安裝node.js 以及 npm<br>
> 並確保為以下版本以上<br>
> node.js v20.12.0<br>
> npm v10.5.0<br>
> 建議使用 root 執行以下所有指令

1.將檔案下載下來
```bash
git clone https://github.com/LucienRay/8591.git
```
2.進入專案目錄
```bash
cd .\8591\
```

3.建置前後端環境
```bash
cd .\Frontend\
npm i
cd ..\Backend\
npm i
```

4.done

## How to run
1.建置前端
```bash
cd .\Frontend\
npm run build
```

2.建置後端
```bash
cd ..\Backend\
npm run build
```

3.啟動後端
```bash
npm run start
```

4.開啟瀏覽器，輸入網址
```
https://localhost
```
